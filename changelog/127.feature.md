Added REST API client, WorkOS OAuth authentication, and backend abstraction layer (`BookshelfBackend` protocol with `S3Backend` and `APIBackend` implementations) for migrating from direct S3 access to API-driven data fetching. `BookShelf` now delegates all data operations through the backend protocol, with `BOOKSHELF_BACKEND=api` env var for opt-in API usage, cache-first loading for pinned versions, and graceful offline fallback to cached data.
